## Installation & Usage

```bash
# 1) Clone repository using `degit`.
$ npx degit joelvoss/jvdx-templates/<template-name> my-package

# 2) Navigate into cloned repository.
$ cd my-package

# 3) Install dependencies.
$ npm install
```
